# Analyze a News Article

This project contains a Python script that analyzes a news article and performs several text-processing tasks as required by the Summative Lab: Analyze a News Article.

## Project goal

The script reads the article text from NewsArticle.txt and provides:

- the number of times a specific word appears
- the most common word
- the average word length
- the number of paragraphs
- the number of sentences

## Implemented functions

- count_specific_word(text, word)
- identify_most_common_word(text)
- calculate_average_word_length(text)
- count_paragraphs(text)
- count_sentences(text)

These functions include the required control structures:

- while loop
- for loop
- if/else statements

## Files

- pythonAssessment.py: main script with the text analysis functions
- NewsArticle.txt: article text used for analysis
- README.md: project documentation

## How to run

From this folder, run:

```bash
python3 pythonAssessment.py
```

You will be prompted to enter a word to count.
